# 🪨 Rock Paper Scissors - React Edition

A modern, fast, and visually stunning Rock Paper Scissors game built with **React** and **Vite**. Challenge the computer, track your wins, and aim for the highest streak!

### 🚀 **[Live Demo](https://rock-paper-scissor-react-coral.vercel.app/)**

---

## ✨ Features

- 🎮 **Interactive Gameplay**: Classic Rock-Paper-Scissors against an automated computer opponent.
- 📊 **Real-time Statistics**: Track your current score, computer score, and total rounds played.
- 🔥 **Streak System**: Keep track of your current win streak and aim to beat your **Max Streak** record.
- 🕒 **Match History**: Review your previous moves and results in a scrollable history list.
- 💎 **Premium UI**: 
  - Glassmorphism design with backdrop filters.
  - Smooth hover effects and micro-animations.
  - Fully responsive layout (Mobile & Desktop friendly).
  - Modern typography using 'Outfit' and 'Poppins' fonts.
- 🔄 **Quick Reset**: Start fresh anytime with the "Play Again" functionality.

---

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Modern CSS3 with Variables & Flexbox)
- **Icons/Emoji**: System Emojis

## 🚀 Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Himanshubansal-2008/Rock-Paper-Scissor-react.git
   cd Rock-Paper-Scissor-react
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

---

## 🎮 How to Play

1. Choose your move by clicking on the **Rock (🪨)**, **Paper (📄)**, or **Scissors (✂️)** icons.
2. The computer will simultaneously make its move.
3. The winner is determined based on the classic rules:
   - **Rock** beats **Scissors**
   - **Scissors** beats **Paper**
   - **Paper** beats **Rock**
4. Watch your streak grow and try to set a new high score!
